Einführung / Introduction

Morpheus ist ein Maubot-Plugin für Matrix, das URLs in Chats automatisch filtert, bewertet und moderiert. Es kombiniert eine anpassbare Blacklist/Whitelist, robusten Anti-Spam-Schutz und automatisierte Moderations-Workflows, um sichere Diskussionen zu gewährleisten.

Zielgruppe: Matrix-Administratoren, Moderatoren und Communities, die zuverlässige URL-Filterung, Missbrauchsschutz und nachvollziehbare Moderationsprozesse benötigen.

Keywords: Matrix, Maubot, URL filtering, Blacklist, Whitelist, phishing, anti-spam, moderation, link-filter, security.

Installation / Installation

DE: Voraussetzungen: Python 3.x, Maubot installiert; Installation via das Plugin-Paket (.mbp) in Maubot; Abhängigkeiten per Pip installieren; Plugin in Maubot konfigurieren; Maubot neu starten.

EN: Prerequisites: Python 3.x, Maubot installed; Install the plugin via the Maubot dashboard (.mbp package); install dependencies with pip; configure the Morpheus plugin; restart Maubot.

Konfiguration / Configuration

DE: Beispiel-Umgebungsvariablen: MORPHEUS_BLACKLIST, MORPHEUS_WHITELIST, MORPHEUS_SECRET_SALT (Pflicht). Weitere Optionen: mod_room_id, command_rooms, enable_link_previews, link_preview_timeout, mute settings, etc.

EN: Example environment variables: MORPHEUS_BLACKLIST, MORPHEUS_WHITELIST, MORPHEUS_SECRET_SALT (required). Other options: mod_room_id, command_rooms, enable_link_previews, link_preview_timeout, mute settings, etc.

Verwendung / Usage

DE: Typischer Ablauf: Morpheus scannt Nachrichten, prüft Domains gegen Blacklist/Whitelist, löst ggf. Shortener-URLs auf, wendet Anti-Spam-Mechanismen an und öffnet Moderations-Workflows. Unknown/Domains landen in der Moderations-Warteschlange; Whitelist-Domains erhalten ggf. eine Link-Vorschau.

EN: Typical workflow: Morpheus scans messages, checks domains against blacklist/whitelist, resolves URL-shorteners when needed, applies anti-spam measures, and triggers moderation workflows. Unknown domains go to the moderation queue; whitelisted domains may receive a link preview.

FAQ

DE: Wie funktioniert Morpheus bei Phishing-URLs?
Morpheus prüft die finale Ziel-URL gegen die Blacklist/Whitelist. Phishing-URLs lösen Moderationsschritte aus.
EN: How does Morpheus handle phishing URLs?
It checks the final URL against the blacklist/whitelist and may trigger moderation actions.
DE: Wie konfiguriere ich Blacklists/Whitelists?
Verwende die Befehle !block / !allow oder bearbeite die entsprechenden Dateien unter data/ (laufende Listen).
EN: How do I configure blacklists/whitelists?
Use the commands !block / !allow or edit the runtime lists under data/ (live lists).
