Introduction

Morpheus is a Maubot plugin for Matrix that automatically filters and moderates URLs in Matrix chats. It combines a customizable blacklist/whitelist, strong anti-spam protections, and automated moderation workflows to keep channels safe.

Audience: Matrix admins, moderators, and communities that require reliable URL filtering, abuse protection, and auditable moderation processes.

Keywords: Matrix, Maubot, URL filtering, Blacklist, Whitelist, phishing, anti-spam, moderation, link-filter, security.

Features

  • Customizable Blacklist/Whitelist for malicious links and phishing URLs.
  • Anti-spam protection with rate limiting and link scanning.
  • Automated moderation workflows (flagging, auto-remove, alerts).
  • Maubot integration for Matrix rooms.
  • Pattern-based URL filtering and extensible rules.
  • Audit logs and event hooks for transparency.
  • URL-shortener resolution (e.g. bit.ly, t.co) – verify final target domains.
  • GDPR-compliant persistence (hashing, 24h retention).

Installation

Prerequisites: Python 3.x, Maubot installed; Install the plugin via the Maubot dashboard (.mbp package); install dependencies with pip; configure the Morpheus plugin; restart Maubot.

Configuration

Example environment variables: MORPHEUS_BLACKLIST, MORPHEUS_WHITELIST, MORPHEUS_SECRET_SALT (required). Other options: mod_room_id, command_rooms, enable_link_previews, link_preview_timeout, mute settings, etc.

Usage

Workflow: Morpheus scans messages, checks domains against blacklist/whitelist, resolves URL-shorteners when needed, applies anti-spam measures, and triggers moderation workflows. Unknown domains go to the moderation queue; whitelisted domains may receive a link preview.

FAQ

How does Morpheus handle phishing URLs?
It checks the final URL against the blacklist/whitelist and may trigger moderation actions.
How do I configure blacklists/whitelists?
Use the commands !block / !allow or edit the runtime lists in data/ (live lists).
