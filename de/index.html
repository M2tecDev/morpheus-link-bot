Einführung / Introduction

Morpheus ist ein Maubot-Plugin für Matrix, das URLs in Chats automatisch filtert, bewertet und moderiert. Es kombiniert eine anpassbare Blacklist/Whitelist, robusten Anti-Spam-Schutz und automatisierte Moderations-Workflows, um sichere Diskussionen zu gewährleisten.

Zielgruppe: Matrix-Administratoren, Moderatoren und Communities, die zuverlässige URL-Filterung, Missbrauchsschutz und nachvollziehbare Moderationsprozesse benötigen.

Keywords: Matrix, Maubot, URL filtering, Blacklist, Whitelist, phishing, anti-spam, moderation, link-filter, security.

Merkmale / Features

  • Anpassbare Blacklist/Whitelist für schädliche Links und Phishing-URLs.
  • Anti-Spam-Schutz mit Ratenbegrenzung und Link-Scanning.
  • Automatisierte Moderations-Workflows (Flaggen, Entfernen, Warnungen).
  • Maubot-Integration für Matrix-Räume.
  • Musterbasierte URL-Filterung und erweiterbare Regeln.
  • Audit-Logs und Event-Hooks für Transparenz.
  • URL-Shortener-Auflösung (z. B. bit.ly, t.co) – finale Ziel-Domains prüfen.
  • DSGVO-konforme Persistenz (Hashing, 24h-Retention).

Installation / Installation

Voraussetzungen: Python 3.x, Maubot installiert; Plugin via Dashboard (.mbp); Abhängigkeiten via pip; Morpheus konfigurieren; Maubot neustarten.

Konfiguration / Configuration

Beispiel-Umgebungsvariablen: MORPHEUS_BLACKLIST, MORPHEUS_WHITELIST, MORPHEUS_SECRET_SALT (erforderlich). Weitere Optionen: mod_room_id, command_rooms, enable_link_previews, link_preview_timeout, mute settings, etc.

Verwendung / Usage

Workflow: Morpheus scannt Nachrichten, prüft Domains gegen Blacklist/Whitelist, löst ggf. Shortener-URLs auf, wendet Anti-Spam-Maßnahmen an und startet Moderations-Workflows. Unbekannte Domains landen in der Moderations-Warteschlange; Whitelist-Domains erhalten ggf. eine Link-Vorschau.

FAQ

Wie funktioniert Morpheus bei Phishing-URLs?
Mor­pheus prüft die finale Ziel-URL gegen die Blacklist/Whitelist; Phishing-URLs lösen Moderationsschritte aus.
Wie konfiguriere ich Blacklists/Whitelists?
Verwende die Befehle !block / !allow oder bearbeite die Laufzeitleisten unter data/.
